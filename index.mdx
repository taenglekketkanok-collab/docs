## Get started in four steps

<Steps>
  <Step title="Create an account">
    Sign up at [google.com](https://google.com) to access the dashboard and generate your credentials.
  </Step>
  <Step title="Generate an API key">
    Navigate to **Settings → API Keys** and create a new key scoped to your needs.
  </Step>
  <Step title="Make your first request">
    Pass your API key in the `Authorization` header and call any endpoint.
  </Step>
  <Step title="Explore integrations">
    Connect webhooks, configure settings, and build your integration using the guides.
  </Step>
</Steps>

<Note>
  New to the platform? Start with the [Quickstart](/quickstart) guide — it walks you through your first working request step by step.
</Note>
