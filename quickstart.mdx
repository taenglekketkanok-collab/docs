This quickstart walks you through everything needed to make your first successful API call against the Google platform. By the end, you'll have a working request that queries the search endpoint and returns results you can build on.

<Steps>
  <Step title="Sign up for an account">
    Go to [google.com](https://google.com) and create a free account. Once registered, log in to access the developer dashboard.
  </Step>
  <Step title="Generate your API key">
    In the dashboard, navigate to **Settings → API Keys** and click **Create Key**. Give it a descriptive name (e.g., `my-first-key`) and set the appropriate scopes.

    <Tip>Store your API key in an environment variable — never hardcode it in source code.</Tip>
  </Step>
  <Step title="Install a client (optional)">
    You can call the API with any HTTP client. For convenience, examples below use `curl`.

    <CodeGroup>
    ```bash curl
    # No installation needed — curl comes pre-installed on most systems
    curl --version
    ```
    ```bash Node.js
    npm install node-fetch
    ```
    ```bash Python
    pip install requests
    ```
    </CodeGroup>
  </Step>
  <Step title="Make your first request">
    Replace `YOUR_API_KEY` with the key you created:

    <CodeGroup>
    ```bash curl
    curl -X GET "https://api.google.com/v1/search?q=hello+world" \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -H "Content-Type: application/json"
    ```
    ```javascript Node.js
    const response = await fetch(
      'https://api.google.com/v1/search?q=hello+world',
      {
        headers: {
          'Authorization': 'Bearer YOUR_API_KEY',
          'Content-Type': 'application/json'
        }
      }
    );
    const data = await response.json();
    console.log(data);
    ```
    ```python Python
    import requests

    response = requests.get(
        'https://api.google.com/v1/search',
        params={'q': 'hello world'},
        headers={'Authorization': 'Bearer YOUR_API_KEY'}
    )
    print(response.json())
    ```
    </CodeGroup>
  </Step>
  <Step title="Inspect the response">
    A successful response returns HTTP `200` with a JSON body:

    ```json Response
    {
      "results": [
        {
          "id": "res_01abc",
          "title": "Hello World",
          "url": "https://example.com/hello",
          "snippet": "A classic introductory phrase used in programming..."
        }
      ],
      "total": 142,
      "page": 1,
      "per_page": 10
    }
    ```
  </Step>
</Steps>
